Para empezar considero indispensable presentarles brevemente en qué consiste el HTML/CSS.

El HTML es un lenguaje de marcado que se utiliza para el desarrollo de páginas de Internet. HTML es la abreviatura de HyperText Markup Language, es decir, Lenguaje de Marcas de Hipertexto, que podría ser traducido como Lenguaje de Formato de Documentos para Hipertexto.

El CSS (hojas de estilo en cascada) es un lenguaje que define la apariencia de un documento escrito en un lenguaje HTML.
